# KitapSepeti QA Otomasyon Projesi

## Proje Hakkında

Bu proje, KitapSepeti e-ticaret sitesi için Cypress kullanılarak geliştirilmiş bir test otomasyon projesidir. Projenin amacı, kullanıcıların en önemli işlemlerini otomatik olarak test etmek ve uygulamanın beklenen şekilde çalıştığını doğrulamaktır.

## Kullanılan Teknolojiler

- Cypress
- JavaScript
- Page Object Model (POM)
- Git & GitHub
- Visual Studio Code

## Test Edilen Senaryolar

- Kullanıcı Girişi (Login)
- Ürün Arama
- Ürün Detay Sayfası
- Sepet İşlemleri
- Checkout (Ödeme Süreci)

## Proje Yapısı

- e2e
- pages
- fixtures
- support

## Test Yaklaşımı

Projede hem pozitif hem de negatif test senaryoları hazırlanmıştır.

Örnek senaryolar:

- Geçerli kullanıcı girişi
- Geçersiz kullanıcı girişi
- Ürün arama
- Ürün detay bilgilerinin doğrulanması
- Sepete ürün ekleme
- Sepet işlemleri
- Ödeme sayfası kontrolleri

## Mimari

Projede kod tekrarını azaltmak, okunabilirliği artırmak ve bakım kolaylığı sağlamak amacıyla **Page Object Model (POM)** yapısı kullanılmıştır.

## Hata Takibi

Manuel testlerde bulunan hatalar Trello üzerinde raporlanmıştır.
